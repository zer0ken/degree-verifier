# `degree-verifier`

이 프로젝트는 마이크로디그리(Microdegree) 프로그램의 이수 여부를 자동으로 검사해주는 Java 기반 애플리케이션입니다.

## 주요 기능

- 학생의 수강 내역 파일(예: Excel, CSV 등) 입력
- 마이크로디그리 이수 조건(필수 과목, 선택 과목, 최소 학점 등) 자동 판별
- 조건 미달 시 부족한 부분 안내
- 결과 리포트 출력
